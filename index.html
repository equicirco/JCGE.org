Julia Computable General Equilibrium Framework

JCGE is a modular, block-based framework for Computable General Equilibrium (CGE) modeling in Julia. This documentation is the project entry point: it describes architecture, workflows, and package boundaries while the package APIs live in their own docs.

Start here

This site is organized around a project overview, a quick start, and a set of cross-package guides. Use the links below to jump to the part of the framework you need.

If you are looking for package-level APIs, start with the Packages page.

  • Project overview: design goals and how the ecosystem fits together
  • Getting Started: install packages and run your first model
  • Packages: what each package does in the ecosystem
  • Guides: modeling, calibration, output, and imports
  • Contact & citation: how to cite and who to reach

Ecosystem at a glance

JCGE is a family of focused Julia packages. The Packages page gives a quick map of the ecosystem and where to look for detailed API references.
